Description:
========
This is android lock overlay plugin using just Javascript. Tested in modern browsers.
How to use:
========
<pre>androidLock(code);</pre>
<b>code</b> -> combination to unlock<br>

Examples of usage:
========
<pre>androidLock("6304852");;</pre>
<br>
<pre>androidLock("012");</pre>
